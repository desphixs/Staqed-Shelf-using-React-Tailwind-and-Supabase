# Build Shelf: A Personal Reading Tracker

## Description
Shelf is a beautiful web app to track every book you read. You will start by setting up a cloud database to store book titles, ratings, and reading progress. Then, you will add a login system so people can create their own accounts. Finally, you will write special security rules to guarantee that when someone logs in, they only see the books on their personal shelf and nobody else can peek at their notes.


## Tech Stack

- React (TypeScript)
- Supabase
- TailwindCSS
